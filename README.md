
# Spandan Gowda — Portfolio

This is a simple, recruiter-focused portfolio website built with static HTML, CSS and a little JS.
It is ready to deploy on **GitHub Pages** or any static hosting (Netlify, Vercel).

## What you get
- index.html, style.css, script.js
- Spandan_Resume.pdf (included from uploaded resume)
- avatar.svg
- README with deploy instructions

## Deploy on GitHub Pages
1. Create a new repo on GitHub (e.g. `spandan-portfolio`).
2. Upload all files from this project to the repository root (index.html etc.).
3. Commit and push to `main` branch.
4. In repository Settings → Pages, set Source to `main` branch and folder `/ (root)`.
5. Your site will be available at `https://<your-username>.github.io/<repo-name>/`

## Customize
- Replace project links in index.html with real GitHub repo URLs.
- Update social links, add screenshots inside an `assets/` folder.
- For a React/Vite version, request 'React upgrade' in chat.
